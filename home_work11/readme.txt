реалізувати графічний інтерфейс для програми з минулого ДЗ 10. Робота з ресурсами, параметрами командного рядка у Qt та робота з конфігураційним файлом

Основні функціональні можливості програми:

    Реєстрація та видалення сенсорів. Користувач може додавати нові сенсори через графічний інтерфейс. Наприклад, кнопка"Додати сенсор". Видалення сенсора також здійснюється задопомогою кнопки "Видалити".
    Симуляція даних сенсорів. для прощення Симуляція може бути виконана для всіх сенсорів з генеруванням випадкових даних. кнопка "Симуляція"

Інтерфейс програми має складатися з 

    списк  сенсовір(для відображення використати QListWidget) 
    3 конки  "Додати сенсор", "Видалити",  "Симуляція" 
    таблиця відображення результатів симуляції. для відображення використати QTableWidget
    QProgressBar для відображення процесу симуляції

кнопка "Додати сенсор" добавляє новий сенсор в списк  сенсовір(QListWidget).

якщо список був пустий - новий сенсор матиме імя: Sensor-1.

Всі наступні нові сенсори матипуть назву на один індекс більший.

кнопка "Видалити" - видаляє активний(віділений) сенсор з списку.

конопка  "Симуляція" запускає симуляцію сенсоірів.

під час симуляції має відображатися прогрес або просто анімуватися робота через віджет QProgressBar
